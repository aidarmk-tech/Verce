# GroupChat PWA — Установка

## Что это?
Приватный мессенджер для вашей группы Telegram (ID: 338845720).
Работает как приложение на Android, iOS и ПК.

---

## Быстрая установка (на телефон)

### Вариант 1: Захостить бесплатно на GitHub Pages

1. Создайте репозиторий на github.com (например `groupchat`)
2. Загрузите ВСЕ файлы из этой папки в репозиторий
3. Зайдите в Settings → Pages → Source: `main` → Save
4. Через минуту сайт будет доступен по адресу:
   `https://ваш-логин.github.io/groupchat/`

### Вариант 2: Захостить на Vercel (ещё проще)

1. Зайдите на vercel.com
2. Import Git Repository → выберите репозиторий
3. Deploy — готово!

### Вариант 3: Локальный сервер (для тестирования)

```bash
# Установите простой HTTP-сервер
npx serve .

# Или через Python
python3 -m http.server 8080
```

Откройте http://localhost:8080 в браузере.

## Установка как приложение

### Android (Chrome):
1. Откройте ссылку в Chrome
2. Нажмите ⋮ (три точки) → **"Установить приложение"**
3. Подтвердите — иконка появится на рабочем столе

### iPhone (Safari):
1. Откройте ссылку в Safari
2. Нажмите 🔗 (Поделиться) → **"На экран «Домой»"**
3. Нажмите "Добавить"

### Windows/Mac (Chrome):
1. Откройте ссылку в Chrome
2. В адресной строке нажмите иконку установки (⊕)
3. Нажмите "Установить"

## Подключение к реальному Telegram

Сейчас приложение работает в демо-режиме. Для реального подключения:

1. Зарегистрируйте приложение на https://my.telegram.org
2. Получите `api_id` и `api_hash`
3. Подключите бэкенд на Python (Telethon) или Node.js (tdl)
4. Замените демо-данные на реальные API-вызовы

Нужна помощь с бэкендом — напишите!
